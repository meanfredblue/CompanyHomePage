Container width: 800px height: 400px

display: flex;

justify-content: flex-start;

align-items: flex-start;

In this example, all boxes have their width and height set to 50px.

Item #1 is set to flex-grow: 1;

Item #2 is set to flex-grow: 2;

Item #3 is set to flex-grow: 3;

Item #4 is set to flex-grow: 4;

Item #5 is set to flex-grow: 5;

The flex-grow value determines what portion of available extra space should be given to each item.

In this example, there are 1 + 2 + 3 + 4 + 5 = 15 total portions that the extra space is divided by.

In other words, to get the number of pixels for a portion, you would divide the total extra pixels by 15.

Then, you would grow each item by the flex-grow value times the size of a portion.

If each item takes up 50px and there are 5 items, the initial used space is 5 X 50 = 250.

If the total width of the container is 800px, then the extra space is 800 - 250 = 550px

The extra portion size is 550/15 = 36.6667

The new wdith of item #1 would be 50px + (1 X 36.6667) = 86.6667 = 87px

The new wdith of item #2 would be 50px + (2 X 36.6667) = 123.3334 = 123px

The new wdith of item #3 would be 50px + (3 X 36.6667) = 160px

The new wdith of item #4 would be 50px + (4 X 36.6667) = 196.6668 = 197px

The new wdith of item #4 would be 50px + (5 X 36.6667) = 233.3335 = 233px

87 + 123 + 160 + 197 + 233 = 800

Container width: 200px height: 400px

display: flex;

justify-content: flex-start;

align-items: flex-start;

In this example, all boxes have their width and height set to 50px.

Item #1 is set to flex-shrink: 1;

Item #2 is set to flex-shrink: 2;

Item #3 is set to flex-shrink: 3;

Item #4 is set to flex-shrink: 4;

Item #5 is set to flex-shrink: 5;

The flex-shrink value determines what portion of overflow width should be subtracted from each item.

The used width is 5 * 50 = 250

The overflow width is 250 - 200 = 50px

The sum of (flex-shrink * flex-basis) for all items = (50 * 1) + (50 * 2) + (50 * 3) + (50 * 4) + (50 * 5) = 750

The shrink ratio for item #1 is (50 * 1)/750

The new width for item #1 is flex-basis - (total to shrink * shrink ratio) = 50 - (50 * (50 * 1)/750) = 46.6667 = 47px

The shrink ratio for item #2 is (50 * 2)/750

The new width for item #2 is flex-basis - (total to shrink * shrink ratio) = 50 - (50 * (50 * 2)/750) = 43.3333 = 43px

The shrink ratio for item #3 is (50 * 3)/750

The new width for item #3 is flex-basis - (total to shrink * shrink ratio) = 50 - (50 * (50 * 3)/750) = 40px

The shrink ratio for item #4 is (50 * 4)/750

The new width for item #4 is flex-basis - (total to shrink * shrink ratio) = 50 - (50 * (50 * 4)/750) = 36.667 = 37px

The shrink ratio for item #5 is (50 * 5)/750

The new width for item #5 is flex-basis - (total to shrink * shrink ratio) = 50 - (50 * (50 * 5)/750) = 33.333 = 33px

47 + 43 + 40 + 37 + 33 = 200
